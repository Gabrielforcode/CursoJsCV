Operadores Relacionais

  1. 5 > 2 == True
  2. 7 < 4 == False
  3. 8 >= 8 == True
  4. 9 <= 7 == False
  5. 5 == 5 == True
  6. 4 != 4 == False

Havendo operadores relacionais e aritméticos na mesma linha, primeiro seram resolvidos os aritméticos

Operadores de identidade

  1. 5 == 5 = True
  2. 5 == "5" = True
  3. 5 === "5" = True
  4. 5 === 5 = True 
        > var x = 5
        undefined
        > var y = '5'
        undefined
        > typeof x
        'number'
        > typeof y
        'string'
        > x == y
        true
        > x === y
        false
        > x != y
        false
        > x !== y
        true

sinal de igualdade ' == 'não testa tipo, já o sinal '===' testa.

Operadores lógicos

  1. ! = não = negação
  2. && = e = conjunção
  3. || = ou = disjunção

! é um operador unário, !true = false e !false = true

&& é um operador binário, if true && true return true else return false (basta que um operador seja falso para que o Js retorne falso)

|| é um operador binário, if false || false return false else return true (basta que um operador seja verdadeiro para o Js retornar verdadeiro)

Primeiro seram respondidos os operadores aritméticos, depois os relacionais e por ultimo os operadores lógicos

A ordem de execução dos operadores lógicos é !, && e por ultimo o ||

Operador ternário

TESTE ? TRUE : FALSE (3 operandos)
exemplo: média >= 7 ? "Aprovado" : "Reprovado"
