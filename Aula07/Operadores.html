Incremento

var x = 5

pós incremento

                    
  1. x = x + 1  : x += 1 :  x++
    2. 
                
  2. x = x - 1  : x -= 1 :  x--
    3.

pré incremento

            
    
                
  1. x = x + 1  : x += 1 :  ++x
    2. 
                
  2. x = x - 1  : x -= 1 :  --x
    3.
