Comentários em JS

JS aceita '//' ou '/* */' para incersão de Comentários. Comandos de comentários só funcionam dentro da tag script

  • // = comentários de uma linha
  • /* */ = comentários de mais de uma linha

Variáveis

Nome de variáveis == identificador

  • Podem começar com letras, $ ou _
  • Não podem comer com números
  • É possível usar letras ou números
  • É possível usar acentos e símbolos
  • Não podem conter espaços
  • Não podem ser palavras reservadas (Não podem ser palavras usadas pelo JS como comandos)

Dicas

  • Maiusculas e minusculas diferem
  • Tente escolher nomes coerentes para as variáveis
  • Evite se tornar um 'programador alfabeto' ou programador 'contador'

Variáves servem para guardar dados sejam eles boolean, number, strings.

Data Types (tipos Primitivos)

  • Number
    • Infinity
    • NaN
  • string
  • boolean
  • null
  • undefined
  • object
    • array
  • function

O comando typeof retorna o tipo da variável, como por exemplo:

        
        > var n = 200
        undefined
        > n
        200
        > typeof n
        'number'
        > var n = 'Google'
        undefined
        > n
        'Google'
        > typeof n
        'string'
        > typeof 6
        'number'
        > typeof 6.5
        'number'
        > typeof "6.5"
        'string'
        > typeof []
        'object'
        > typeof {}
        'object'
        > typeof function () {}
        'function'
        > typeof undefined
        'undefined'
        > typeof NaN
        'number'
        > typeof Infinity
        'number'
        > typeof null
        'object'
