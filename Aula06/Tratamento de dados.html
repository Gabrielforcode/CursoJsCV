Para converter de string para number utizamos

  • Number.parseInt (n) (Converção para Number inteiro)
  • Number.parseFloat (n) (Converção para Number real)
  • Number (n) (converte conforme valor informado)

Para convertemos de number para string utilizamos

  • String(n)
  • n.toString()

Formatação de String

  • s.length retorna o tamanho de s
  • s.toUpperCase() retorna s em caixa alta
  • s.tolowerCase() retorna s em caixa baixa

Formatação de Numeros

  • s.toFixed(n) para adicionar casas decimais
  • s.toFixed(n).replace('.' , ',') pra subistituir . por ,
  • s.toLocaleString('pt-BR' {style: 'currency', currency: 'BRL'})
  • s.toLocaleString('pt-BR' {style: 'currency', currency: 'USD'})
