Oque é DOM?

DOM significa Document Object Model
É o conjunto de objetos dentro do navegador que darão acesso aos objetos internos do site. O DOM está presente quando usamos JS dentro do navegador

Arvore DOM

Metodos de acesso

por..

  • por marca (getElementsByTagName)
  • por ID (getElementsById)
  • por Nome (getElementsByName)
  • por Classe (getElementsByClassName)
  • por Seletor querySelector()
